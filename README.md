# Tola (ADHDAssist)

A playful and feature-rich task management application designed to help users with ADHD stay organized, motivated, and productive.

## 🚀 Recent UI Modernization

The app has recently undergone a major UI overhaul to provide a more consistent, engaging, and professional experience:

- **Playful Theme:** A new color palette featuring "Softer Skyblue", "Softer Gold", and pastel status indicators.
- **Custom Typography:** Integration of `BubblegumSans-Regular` and `RubikBubbles-Regular` fonts for a friendly, approachable feel.
- **Card-Based Layout:** Tasks are now displayed in polished cards with priority-coded borders (Red for High, Yellow for Medium, Green for Low).
- **Redesigned Forms:** Add and Update task screens now feature a clean, card-based structure with improved attachment management.
- **Enhanced Navigation:** A cohesive experience across the Home tab, Mission Control (TaskList), Mission History (Completed Tasks), and the Pomodoro Timer.

## ✨ Core Features

- **Mission Control (Task Management):** Add, update, and track tasks with status updates (Pending, Progress, Done).
- **Visual Schedule:** An integrated Agenda calendar view for planning your day.
- **Mission History:** Track your productivity with a categorized view of completed missions.
- **Pomodoro Timer:** Stay focused with a custom timer, including music support and focus settings.
- **Gamification:** Unlock a special Game screen by completing daily tasks.
- **Attachments:** Support for adding photos and files to your tasks.
- **Authentication:** Secure login and registration powered by Firebase.

## 🛠️ Technical Stack

- **Framework:** React Native (Expo)
- **Navigation:** React Navigation (Stack, Tab, Drawer)
- **Database/Auth:** Firebase (Firestore & Auth)
- **UI Components:** Expo Vector Icons, React Native Calendars, Gifted Charts
- **Styling:** Custom Theme System (`utils/theme.js`)

## 🚦 Getting Started

### Prerequisites
- Node.js
- Expo Go app on your mobile device (or an emulator)

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the project:
   ```bash
   npm start
   ```
4. Scan the QR code with Expo Go (Android) or the Camera app (iOS).
